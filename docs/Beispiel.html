Bedienungsanleitung TShare

TShare ist ein webbasiertes Journal, das zum Erstellen unterschiedlichster HTML-Inhalt im Browser dienen kann. Zu diesem Zweck werden verschiedene Module zur Verfügung gestellt:

  • Handschriftliche Eingabe (Speicherung als SVG-Bild)

  • Einfügung von PDFs (Speicherung als PNG-Bild)

  • Auf Asciidoctor basierendes Textmodul mit zusätzlicher Unterstützung zur Eingabe mathematischer Ausdrücke in Form von Latex-Formeln und einer vereinfachten Eingabe über den AsciiMath-Syntax. Gerendert werden mathematische Ausdrücke von KaTeX

  • Einfügen von Bildern

== Bedienungsanleitung TShare TShare ist ein webbasiertes Journal, das zum Erstellen unterschiedlichster HTML-Inhalt im Browser dienen kann. Zu diesem Zweck werden verschiedene Module zur Verfügung gestellt: * Handschriftliche Eingabe (Speicherung als SVG-Bild) * Einfügung von PDFs (Speicherung als PNG-Bild) * Auf http://asciidoctor.org[Asciidoctor] basierendes Textmodul mit zusätzlicher Unterstützung zur Eingabe mathematischer Ausdrücke in Form von https://de.wikipedia.org/wiki/LaTeX[Latex-Formeln] und einer vereinfachten Eingabe über den http://asciimath.org/[AsciiMath-Syntax]. Gerendert werden mathematische Ausdrücke von https://khan.github.io/KaTeX/[KaTeX] * Einfügen von Bildern

Eine Besonderheit besteht darin, dass die Erstellung komplett im Browser abläuft und das erstellte Produkt eine von jedem Browser darstellbare HTML-Datei ist. Diese kann nur mit den Inhalten, bei Bedarf aber auch inklusive des eingebetteten Programms, veröffentlicht werden (z. B. auf Moodle).

=== Texteingabe

Die Live-Vorschau ist standardmäßig eingestellt und rendert den aktuellen Text nach jeder Betätigung der Leer- oder Eingabetaste.

Insbesondere beim Einbetten von externen Objekten ist es sinnvoll diese Funktion abzuschalten, da das Programm sonst nach jedem Drücken der Leertaste versucht, z. B. ein Video erneut einzubinden.

Für solche Fälle ist der Vorschau-Knopf hilfreich, der nur bei Bedarf die Seite neu darstellt.

Mathematik kann im AsciiMath- oder Latex-Format eingegeben werden.

AsciiMath wird über die Eingabe von

stem:[3/4]

latexmath:[\frac34]

so führt dies wieder zu 34\frac34​4​​3​​, allerdings diesmal im LaTeX-Format eingegeben.
